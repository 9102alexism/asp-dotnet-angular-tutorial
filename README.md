# ASP.NET Core & Angular Tutorial

A simple tutorial project demonstrating how to build a web app with ASP.NET Core backend and Angular frontend.

## Features
- ASP.NET Core API
- Angular SPA
- Entity Framework Core with SQLite
- RESTful endpoints
